# immunforsvar
Immunforsvaret som Middelalderborg
Dette prosjektet er en visuell og interaktiv modell som forklarer immunforsvaret gjennom en middelalder-analogi.

Konsept
Immunforsvaret er komplekst. Ved å sammenligne biologiske komponenter med en borgs forsvar, blir det lettere for elever å forstå samspillet mellom det medfødte og det tilpassede immunforsvaret.

Mapping-oversikt

• Borgmuren: Huden (Fysisk barriere)

• Vollgraven: Slimhinner og magesyre

• Speidere: Dendrittiske celler (Antigenpresentasjon)

• Generalen: T-hjelpeceller (Koordinering)

• Våpensmien: B-celler (Produksjon av antistoffer)

• Snikmordere: T-morderceller (Eliminerer infiserte celler)

Teknisk oppsett
Prosjektet bruker en sentralisert `data.json` for å styre innholdet i den visuelle modellen. Dette gjør det enkelt å oppdatere fakta eller legge til nye roller.

Installasjon
1. Clone repoet.

2. Åpne `index.html` i nettleseren (eller kjør via en lokal server).
